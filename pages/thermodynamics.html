Термодинамика

Раздел физики, изучающий тепловые явления и превращения энергии. Термодинамика описывает взаимосвязь между теплотой, работой и внутренней энергией систем, устанавливая фундаментальные законы, которые управляют всеми тепловыми процессами.

Законы термодинамики

Первый закон

Закон сохранения энергии: изменение внутренней энергии системы равно сумме полученного тепла и совершенной над системой работы.

Второй закон

Невозможность создания вечного двигателя второго рода. Энтропия изолированной системы не убывает.

Третий закон

При приближении температуры к абсолютному нулю энтропия любой системы стремится к постоянному минимуму.

Нулевой закон

Если две системы находятся в тепловом равновесии с третьей, то они находятся в тепловом равновесии друг с другом.

Основные формулы термодинамики

Формула Описание Обозначения
ΔU = Q - A Первый закон термодинамики ΔU — изменение внутренней энергии, Q — полученное тепло, A — совершенная работа
Q = c·m·ΔT Количество теплоты при нагревании c — удельная теплоемкость, m — масса, ΔT — изменение температуры
pV = νRT Уравнение состояния идеального газа p — давление, V — объем, ν — количество вещества, R — газовая постоянная, T — температура
A = p·ΔV Работа газа при изобарном процессе p — давление, ΔV — изменение объема
η = (Q₁ - Q₂)/Q₁ КПД тепловой машины η — коэффициент полезного действия, Q₁ — полученное тепло, Q₂ — отданное тепло
ΔS ≥ Q/T Изменение энтропии ΔS — изменение энтропии, Q — теплота, T — температура

Термодинамические процессы

Изопроцессы

Изотермический процесс (T = const): происходит при постоянной температуре. Для идеального газа описывается законом Бойля-Мариотта: pV = const.

Изобарный процесс (p = const): происходит при постоянном давлении. Опиывается законом Гей-Люссака: V/T = const.

Изохорный процесс (V = const): происходит при постоянном объеме. Опиывается законом Шарля: p/T = const.

Адиабатный процесс: происходит без теплообмена с окружающей средой (Q = 0). Используется в описании циклов двигателей внутреннего сгорания.

Тепловые машины

Тепловые машины преобразуют тепловую энергию в механическую работу. Наибольшее теоретическое значение КПД имеет цикл Карно, который определяется формулой: η = 1 - T₂/T₁, где T₁ — температура нагревателя, T₂ — температура холодильника.
